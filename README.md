# Kalkulator Matriks

Program Java untuk menyelesaikan SPL dengan Eliminasi Gauss, Inverse, dan Kaidah Cramer serta melakukan Interpolasi Polinom, Interpolasi Bicubic Spline, dan Regresi Linear dan Kuadratik Berganda.

## Technologies Used
- Java

## Feature
- Penyelesaian SPL dengan Eliminasi Gauss, Gauss-Jordan, Inverse, dan Kaidah Cramer
- Menghitung determinan dengan metode Kofaktor dan Inverse
- Menghitung Inverse Matrix dengan Metode Reduksi Baris dan Adjoin
- Melakukan Interpolasi Polinomial
- Melakukan Interpolasi Bicubic Spline
- Melakukan Regresi Linear dan Kuadratik Berganda
## How to Use
### Dependencies
- Java Virtual Environment
- Java Development Kit
### Installation
- Download and install Java
- Download and install Java Development Kit
- Download all folder and files on this repository or simply clone this repo
